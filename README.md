# Music Application

A modern, full-stack **Django-based music streaming platform** for discovering songs, creating playlists, and enjoying a smooth audio experience.

---

##  Features

*  Stream and play music seamlessly
*  Trending songs & new releases
*  Create and manage personal playlists
*  Artist & song discovery
*  Recently played history
*  Keyboard shortcuts for playback control

---

##  Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite / PostgreSQL
* **UI:** Custom responsive design

---

##  Getting Started

```bash
git clone https://github.com/your-username/music-app.git
cd music-app
pip install -r requirements.txt
python manage.py runserver
```
